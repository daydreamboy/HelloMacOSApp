## 1、介绍

MacOS app template project for multiple demos



## 2、使用方式

新建一个MacOS app工程，copy下面文件到新工程。如果新工程，已有问题，则覆盖文件

```shell
$ tree .
.
├── AppDelegate.swift
├── Base.lproj
│   └── MainMenu.xib
├── Classes
│   ├── Demo1WindowController.swift
│   └── Demo2WindowController.swift
└── UI
    ├── Demo1WindowController.xib
    └── Demo2WindowController.xib

```
